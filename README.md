# 💧 IoT-Based Smart Water Quality Monitoring System

An IoT project that monitors water quality using ESP32, pH, turbidity, and temperature sensors with a responsive web dashboard.

## Features
- 🔹 Real-time temperature monitoring
- 🔹 pH monitoring
- 🔹 Turbidity monitoring
- 🔹 User login and registration
- 🔹 Admin dashboard
- 🔹 Analytics and alert history

## Technologies Used
- HTML
- CSS
- JavaScript
- ESP32
- Arduino IDE
- ThingSpeak
