About Project



Smart Water Quality Monitoring & Alert System

This project is an IoT based Smart Water Quality Monitoring and Alert System developed using ESP32 microcontroller and multiple sensors.


The system continuously monitors important water quality parameters and sends the sensor readings to ThingSpeak cloud platform for real-time monitoring.


Components Used

  • ESP32 Microcontroller
  • DS18B20 Temperature Sensor
  • Turbidity Sensor
  • pH Sensor
  • ThingSpeak Cloud Platform
  • Web Dashboard

Features

  • Real-time water quality monitoring
  • Live dashboard updates
  • Cloud data storage
  • Analytics graphs
  • Water safety status detection
